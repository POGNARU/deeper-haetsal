# deeper-haetsal

> 디퍼(deeper) **언브랜딩 — 햇살** 과제를 모아두는 **공개 아카이브**입니다.

포근나루(POGNARU STUDIO)의 햇살(이다현)이 디퍼 언브랜딩 살롱에서 진행한 과제와 정리물을 한자리에 쌓아 두는 저장소입니다.
처음에는 '나를 알아가는 기록'으로 시작했지만, 지금은 **디퍼 언브랜딩-햇살 과정의 과제 아카이브**로 용도를 옮겼습니다.

## 무엇을 담나

- 디퍼 언브랜딩-햇살 과정에서 나온 과제 결과물
- 과제별 정리·기록

> 지금은 README만 둔 빈 저장소입니다. 과제가 쌓이는 대로 이곳에 더해 갑니다.
>
> 디퍼 살롱 자료 별자리 목차는 별도 공개 레포 **[deeper-index](https://github.com/POGNARU/deeper-index)** ( https://pognaru.github.io/deeper-index/ )로 옮겼습니다.
